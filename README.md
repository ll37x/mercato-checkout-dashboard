# Mercato OpsCenter: Checkout Health Dashboard

## Deliverable 1: Running the Application Locally
This project was bootstrapped with React and Vite for maximum performance and easy setup.

1. Ensure you have Node.js installed.
2. Clone or unzip this repository.
3. Open your terminal and navigate to the project directory.
4. Run `npm install` to install dependencies (Recharts, Tailwind, Lucide).
5. Run `npm run dev` to start the local development server.
6. Open the provided `localhost` URL (usually `http://localhost:5173`) in your browser.
